# go-examples
Examples of various Go constructs

## to run
clone repo

cd into folder

run code
```shell
git clone https://github.com/forrest321/go-examples.git
cd go-examples
```
Then run with Go
```shell
go run . //need to run the whole directory. go run main.go will not work
```
or
Build and run
```shell
go build
./go-examples
```
